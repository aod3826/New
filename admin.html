LINE Messaging API

LINE Notify ถูกปิดให้บริการแล้ว (ตั้งแต่ 1 เม.ย. 2025)
กรุณาใช้ LINE Messaging API ด้านล่างนี้แทน

วิธีรับค่า LINE Messaging API:
  1. ไปที่ LINE Developers Console
  2. สร้าง Provider และ Channel (Messaging API)
  3. ไปที่ Basic settings > คัดลอก Channel Secret
  4. ไปที่ Messaging API > คัดลอก Channel Access Token (Long-lived)
  5. เพิ่ม LINE Official Account เป็นเพื่อน แล้วรับ Group ID จาก Webhook
  6. สร้าง LIFF app และคัดลอก LIFF ID

ใส่ Group ID เพื่อรับแจ้งเตือนออเดอร์ใหม่

ใส่ LIFF ID สำหรับให้ลูกค้า Login

ยังไม่ได้ตั้งค่า LINE Messaging API

จำเป็นต้องเปลี่ยนรหัสผ่าน
